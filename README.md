👁️ Face Recognition System

📌 Description

This project is a Face Recognition System that can detect and recognize human faces using a webcam or images. It uses computer vision and machine learning techniques to identify individuals based on facial features.

🚀 Features

Detect faces in real-time
Recognize known faces
Store and match face data
Works with webcam and images
Fast and efficient processing

🛠️ Technologies Used

Python
OpenCV
NumPy
face_recognition library

📂 Project Structure
Face-Recognition/
│── dataset/           # Images of known people  
│── encodings/         # Saved face encodings  
│── main.py            # Main program  
│── requirements.txt   # Required libraries  
│── README.md          # Project documentation  

⚙️ Installation
Clone the repository:
git clone https://github.com/your-username/face-recognition.git
cd face-recognition
Install dependencies:
pip install -r requirements.txt

▶️ Usage
Run the program:
python main.py
Press Q to exit
The system will detect and recognize faces in real-time

📸 How It Works
Capture face images
Convert faces into numerical data (encodings)
Compare with stored data
Display name if matched
